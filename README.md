# DocuMindAI 🚀

DocuMindAI is a robust Retrieval-Augmented Generation (RAG) pipeline that transforms technical PDFs into intelligent, searchable knowledge. Built with LangChain, ChromaDB, and Groq, it features a modular architecture with LangSmith tracing for real-time observability. Designed for accuracy, it optimizes document chunking and retrieval to minimize hallucinations in LLM responses.

## Features
- **Intelligent Ingestion:** Advanced PDF processing with recursive character splitting.
- **Persistent Storage:** Vectorized storage using ChromaDB for high-speed retrieval.
- **Observability:** Integrated with LangSmith to trace and debug the RAG pipeline.
- **Model Flexibility:** Configurable architecture to swap LLMs without refactoring.

## Tech Stack
- **Framework:** LangChain
- **LLM:** Groq (Llama-3.3-70B)
- **Vector DB:** ChromaDB
- **UI:** Streamlit
- **Monitoring:** LangSmith
